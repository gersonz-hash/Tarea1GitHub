# Tarea1GitHub
Mi opinión sobre GitHub for Students es que es una gran oportunidad para nosotros los estudiantes, porque si nos damos cuenta GitHub tiene muchos beneficios y nos 
da acceso a muchas herramientas que tienen un costo y gracias a la cuenta de la UMG tenemos acceso a GitHub y sus demás funciones tambíen nos ayuda mucho para que 
aprendamos a programar. Lo que yo veo es que GitHub Pro es muy útil porque crea repositorios privados y organiza mejor nuestros proyectos otra cosa que también me
llamo la atención es que es una forma en la que podemos facilitar nuestro trabajo y obtener nuevos conocimientos, para mejorar nuestras habilidades de programación
en un futuro.

Herramientas que me llamaron la atención:
GitHub Foundations Certificaction: Me llamo la atención porque esto nos puede ayudar como futuros programadores es un certificado que nos da sobre que ya sabemos usar
la herramienrta de Git y GitHub con el fin de que vean que ya sabeos usar estos dos anteriores.

Intro to Copilot: Este me llamo mucho la atención porque mientras estamos escribiendo código lo que hace esta herramienta es que nos sugiere código mientras estamos 
escribiendo y nos da ideas cuando no sabemos como empezar el proyecto que estemos haciendo, es como un asistente de inteligencia artificial. 

En este caso aún no tengo activo mi GitHub Pro porque lo solicite y no me lo han activado pero investigue sobre que beneficios tiene y que programas nos permite 
utilizar de forma gratuita.
